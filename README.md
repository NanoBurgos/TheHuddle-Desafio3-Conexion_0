# TheHuddle-Desafio3-Conexion_0

# ¿Quién sos después de este reto? 
Alguien que sabe por que y para que es cada parte del sistema
Alguien que ya no se asusta cuando ve mensajes de Error
Alguien que aprendio a unir hardware y software para llevar adelante un proyecto
Alguien que entendio que un proyecto real no es lineal, es Caos puro 
Y que leyendo los errores que van surgiendo luego de cada modificacion 

# ¿Cómo sobrevivió tu aplicación? 

¿Sobrevivio? 

Luego de aprender sobre bloqueos, condiciones de carrera y manejo errores
Luego de entender sobre el ciclo de vida de un servidor, sobre conexiones
Luego de entender a los hilos daemon y no daemon
y Luego de que aprendi a manejar una gran cantidad de errores, recien ahi puedo decir que mi app sobrevivio

Creo que sobrevivio por que aprendi como hacer que maneje muchos de los errores que fueron surgiendo mientras estaba programando


# ¿Qué aprendiste cuando todo se rompió?
Aprendi que equivocarse es parte natural del desarrollo
Aprendi a prestarle mucha mas atencion a los mensajes de Error
Aprendi sobre la importancia sobre los hilos cuando hay operaciones bloqueantes


Aprendi que desarrollar este tipo de proyectos no se trata de que nada falle, sino a entender del por que fallo y asi poder solucionar el error
Aprendi que los errores no frenan el proyecto, nos muestran el camino
